<Tag label="VIP 客户" variant="brand" />

## API

| Prop | Type | 默认 | 说明 |
|---|---|---|---|
| `label` | `string` | — | 标签文字 |
| `variant` | `'brand' \| 'success' \| 'error' \| 'info' \| 'neutral' \| 'outline'` | `'neutral'` | 语义变体 |
| `size` | `'md' \| 'lg'` | `'md'` | 高度 22 / 26 |
